Overview of the game: 
This almost original idea, not resempeling anything alike obe of the best selling games ever Subway Surfers, is an endless runner. The game consists of the player character running along a
three lane row of persian rugs in a thrid person view. The player needs to dodge multiple differnet types of obstacles to progress in the game. The goal of the game is to run as far along 
the rows of persian rugs to get the highest possible score. If the player collides with a red obstacle the game ends with a gameover panel, which shows the final score. 


Running it:
Download Unity >= 2021.2.14f
Clone or Download the project
The game requires a computer with a mouse and keyboard
